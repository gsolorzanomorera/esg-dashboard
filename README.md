Overview & Approach
This document provides a structured solution to the ESG Dashboard design activity. 
For each company, you will find: (1) a full annotated dataset with data quality flags, (2) recommended dashboard KPIs with prioritisation rationale, (3) a proposed dashboard layout narrative, and (4) critical data integrity notes for the presentation rubric.

Key design principle:
Desing your ESG dashboard as a decision-support tool. 
Every metric selected must directly serve the specific audience's question. Priority data quality issue must be flagged, not buried.
